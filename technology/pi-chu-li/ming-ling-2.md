echo 输出/回显

```
#关闭/开启回显
echo off/on 

#输出
echo list
```

#### 命令行参数
批处理支持命令行参数的概念，其参数可以再被调用时传递给批处理文件。参数可以通过变量%1,%2,%3等从批处理文件调用
```
@echo off
echo %1
echo %2
echo %3
```
将上面存储为文件,并运行

```
C:\Users\Administrator\Desktop>a.cmd 1 2 3
#结果
1
2
3
请按任意键继续. . .
```
